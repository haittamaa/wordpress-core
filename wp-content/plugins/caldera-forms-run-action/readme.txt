== Description ==
A free add-on for [Caldera Forms](https://Calderawp.com/downloads/caldera-forms?utm_source=dotOrg&utm_medium=plugins&utm_campaign=cf-run-action). A simple, free form processor that lets you do anything with your form data, by specifying an action to run on form submission.

Use Caldera Forms to create a responsive user interface for your WordPress site or application, do what ever you want with the data. [Learn more here](https://calderawp.com/downloads/caldera-forms-run-action?utm_source=dotOrg&utm_medium=plugins&utm_campaign=cf-run-action).

== Installation ==
Install & Activate the plugin through the 'Plugins' menu in WordPress
